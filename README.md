## ✨ Features

- ➕ **Basic Operations** - Addition, subtraction, multiplication, division
- ⌨️ **Keyboard Support** - Use your keyboard for all operations
- 🖱️ **Mouse Support** - Fully clickable interface
- 🎨 **Modern Design** - Dark theme with gradient accents
- 📱 **Responsive** - Works perfectly on all devices
- 🔄 **Clear & Backspace** - Easy correction of mistakes
- ⚡ **Real-time Display** - See calculations as you type
- 🧮 **Decimal Support** - Handles floating point numbers

## 🎮 How to Use

### Mouse Controls
- Click number buttons (0-9) to enter digits
- Click operation buttons (+, -, ×, ÷) for calculations
- Click `=` to see result
- Click `C` to clear everything
- Click `⌫` to delete last character
- Click `.` for decimal point

### Keyboard Controls
| Key | Action |
|-----|--------|
| `0-9` | Enter numbers |
| `+` `-` `*` `/` | Operations |
| `Enter` or `=` | Calculate |
| `Backspace` | Delete last character |
| `Escape` | Clear all |
| `.` | Decimal point |
